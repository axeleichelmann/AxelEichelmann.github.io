Python Titanic Survivors Prediction Model

This solo project was completed as part of the famous 'Titanic - Machine Learning from Disaster' Kaggle competition. The aim of the project was to predict whether a set of passengers on the Titanic survived or not based on various pieces of information such as their age, class, gender, cabin number, number of family members on board, etc.
My final model was able to achieve an overall accuracy of 78.9%.

Skills Demonstrated: Python, Exploratory Data Analysis, Feature Engineering, Logistic Regression, SVC, Random Forest Classifier, Ensemble Modelling, Model Evaluation, GridSearchCV (Model Tuning), ML Pipelines, Oversampling, Data Preprocessing
